# ngTicTacToe

✔️ Utilisation de Socket.IO<br/>
✔️ Angular<br/>

Morpion "multiplayer" en ligne en Angular et avec Socket.IO.

### 📕 Pour deployer l'application localement:

Dans une console...
Si besoin installer angular CLI.

``` 
npm install -g @angular/cli
``` 

``` 
cd client
npm install
ng serve --open
```

``` 
cd server
npm install
node app.js
```
